## 2026-07-02 distilled update from PR #176

PR #176 adds a large planted-packet/verifier branch.  The reusable part is the
following deficiency-one chart theorem; the full 3k-line verifier and declared
planted packets are intentionally not merged here until they are split into a
smaller proof artifact.

Let `t=A-k`, `j=n-A`, and specialize to the first underdetermined boundary
`t=j`.  Write the Hankel split as an affine pencil

```text
M(Z)=M0+Z M1
```

of shape `j x (j+1)`.  For `0 <= i <= j`, let `M_i(Z)` be the maximal minor
obtained by deleting column `i`, set `c_i(Z)=(-1)^i M_i(Z)`, and define the
Cramer locator

```text
L_Z(X)=sum_i c_i(Z) X^i.
```

If the pencil is not identically rank-deficient, every finite slope falls into
one of three charts:

1. **Rank drop:** all `c_i(z)` vanish.  The slope is in the common zero locus
   of the maximal minors; each such minor has degree at most `j`.
2. **Full-rank low-degree:** rank is `j` but `c_j(z)=0`.  The one-dimensional
   kernel is generated by a locator of degree `<j`, so any split locator is
   charged to a higher-agreement bucket rather than counted as new exact-`A`
   mass.
3. **Full-rank top chart:** rank is `j` and `c_j(z) != 0`.  The kernel is
   spanned by the degree-`j` locator `L_z`.  Since `H` is the full root set of
   `X^n-1` and `char(F)` does not divide `n`, the locator is valid exactly when
   `L_z(X) | X^n-1`.  On the top chart this is cut out by pseudo-remainder
   coefficients whose slope-degree is at most `(n-j+1)j`.

The generic rank-drop branch admits a finite test.  If
`ker M(z0)=span(P,Q)`, remove the common base roots
`B0={h in H : P(h)=Q(h)=0}` and study the projective evaluation map

```text
h -> [P(h):Q(h)] in P^1(F).
```

A split degree-`j` locator exists exactly when some fiber has size
`j-|B0|`, with nonzero top coefficient.  The contained rank-one case
`ker M(z0)=A(X)F_{<=1}[X]`, with split `deg A=j-1`, is not a new residual:
all degree-`j` locators are `A(X)(X-r)` and are charged to agreement `A+1`.

The additional rank-drop converse proved in this follow-up removes the
remaining valid-split rank-drop branch from the new exact-`A` ledger.  Suppose
`L_R(X)=prod_{r in R}(X-r)` is a valid split degree-`j` locator in `ker M(z0)`.
The first `2j` moments are then uniquely represented on `R` by Vandermonde
interpolation.  If `rank M(z0)<j`, the leading Hankel factorization
`V_R diag(w_r) V_R^T` is singular, so some interpolation weight vanishes.  The
locator on the active roots has degree `<j` and also annihilates the same
moment window.  Thus the slope is charged to a higher-agreement bucket and is
not new exact-`A` mass.  Rank-drop slopes without a valid split degree-`j`
locator are irrelevant to `B_mca(A)` by definition.

For the current row at `A=384`,

```text
t=j=128,
n-j+1=385,
top pseudo-remainder degree cap = 49280,
rough resultant degree cap = 65536,
rank-drop/low-degree minor degree cap = 128,
generic rank-drop local degree cap = 385.
```

Thus the useful M5 output is not a leaderboard row.  It is a precise atlas for
the first underdetermined bucket: the rank-drop valid-split branch dedupes to
higher agreement, so the remaining hard finite branch is the full-rank top
pseudo-remainder root table.  This is directly relevant to the `B_mca(a)`
staircase in `tex/towards-prize.tex`, but it does not yet prove a threshold
bound.

The full-rank top chart now has its finite validity filter proved.  If
`rank M(z)=j` and the top Cramer coefficient `c_j(z)` is nonzero, then
`ker M(z)` is generated by the degree-`j` locator

```text
L_z(X)=sum_i c_i(z)X^i.
```

Because `H` is the full root set of `X^n-1` and `char(F)` does not divide `n`,
the locator is split with `j` distinct roots in `H` if and only if

```text
L_z(X) | X^n - 1.
```

Equivalently, after normalizing by the nonzero top coefficient, the top chart
is cut out by the pseudo-remainder of `X^n-1` on division by `L_z`.  A
full-rank top-chart slope whose Cramer locator does not divide `X^n-1` is not
bad at exact agreement `A`.

The verifier also includes a declared affine toy family over `F_97`.  Two
planted split moment windows `W0,W1` on an order-8 subgroup are joined by

```text
W(z)=W0+z(W1-W0).
```

For every `z in F_97`, the verifier computes the deficiency-one Cramer locator,
classifies rank-drop and low-degree slopes, and applies the top-chart
pseudo-remainder test.  The resulting root table is exactly

```text
z in {0,1},
```

and these two slopes recover the two planted split root sets.  This is only a
toy declared-family root table; it does not claim the corresponding F17 table.

The same toy family also verifies the root-containment certificate logic in
symbolic form.  The script computes the Cramer locator over `F_97[z][X]`,
pseudo-divides `X^8-1` by it, and selects one nonzero pseudo-remainder
coefficient.  That coefficient has degree `19 <= 20`, where `20=(8-4+1)4` is
the toy degree cap, and its zero set contains the root table `[0,1]`.  Thus the
general packet logic is:

```text
if one pseudo-remainder coefficient Q(z) is nonzero, then every full-rank
top-chart bad slope is a root of Q;
if all pseudo-remainder coefficients vanish identically, record an identity
chart/residual rather than claiming a finite root table.
```

The same declared family now verifies the canonical gcd version of this
certificate.  Let `R_i(z)` be the nonzero pseudo-remainder coefficients and set

```text
G_raw(z)=gcd_i R_i(z).
```

Every full-rank top-chart bad slope is a root of `G_raw`.  The raw gcd may also
contain low-degree boundary factors coming from the inequation `c_j(z) != 0`;
therefore the top-chart eliminant is the saturation of `G_raw` away from the
top Cramer coefficient `c_j(z)`.  In the toy `F_97` family, the raw gcd has
roots `[0,1,75]`, the top boundary has roots `[46,75]`, and the saturated gcd
has exactly the root table

```text
z in {0,1}.
```

This is the cleaner object for the eventual F17 top-chart branch: not one
chosen coefficient, but the top-coefficient-saturated gcd of all
pseudo-remainder coefficients.

## The bucket

For exact agreement `A`, in the convention of the on-main extractor
(`extract_regular_hankel_minors.py`: rows `range(t)`, cols `range(j+1)`, entry
`S[row+col]`, `S` = syndrome window of `u + Z*v`), the Hankel system is

```text
t x (j+1),   t = A - k equations,   j + 1 = n - A + 1 locator coefficients,
regular root-containment certificate available  <=>  t >= j+1  <=>  2A >= n+k+1 = 769.
```

**A=384 is the maximal underdetermined agreement:** `t = j = 128`, matrix
`128 x 129`, rank `<= 128 < 129`, so the kernel is nontrivial for **every**
slope `Z` — kernel-nonemptiness (which for `A >= 385` forces `Delta_A(Z) = 0`
and yields the degree-`(j+1)` root-containment bound) certifies **nothing**
here. This is the first singular bucket in the precise M5 sense: the
underdetermined branch begins at exactly this agreement, with **deficiency**
`(j+1) - t = n + k + 1 - 2A = 1`, growing to 239 at the hard-core instance
`A=265`. The underdetermined range with `t >= 1` is `257 <= A <= 384`
(128 exact agreements — the entire near-capacity middle where the open prize
content lives).

## Why deficiency 1 is the right first target

At deficiency 1 the kernel is generically one-dimensional, and is spanned by
the **Cramer vector of signed maximal minors**:

```text
c_i(Z) = (-1)^i * M_i(Z),   i = 0..j,
```

where `M_i(Z)` is the `t x t` minor of `M_A(Z) = H(u) + Z*H(v)` omitting column
`i` — each a polynomial of degree `<= t = 128` in `Z`. So the candidate locator
is a **bivariate polynomial**

```text
L_Z(X) = sum_i c_i(Z) X^i,
```

and a slope can only be bad at exact agreement `A` if its kernel contains a
**valid** locator: degree exactly `j`, splitting over the domain `H` with `j`
distinct roots. Since `H = {x : x^512 = 1}` (the full 2-Sylow subgroup) and
`deg_X L_Z = j = 128`, validity of the generic-kernel locator is equivalent to
the **divisibility filter**

```text
L_Z(X)  |  X^512 - 1        (in F[X], for that value of Z),
```

which is an algebraic condition on `Z` alone: the remainder coefficients of
`X^512 - 1` mod `L_Z(X)` (pseudo-division to stay polynomial), or equivalently
`Res_X(L_Z(X), X^512 - 1) = 0` plus separability. Any nonzero polynomial in the
ideal generated by these conditions is an **eliminant** `Q(Z)` for the
generic-kernel chart of this bucket — the first eliminant-type object at any
underdetermined agreement of this row. The complementary charts are now:
low-degree kernels (`c_j(Z) = 0` — the "projective infinity" analogue in
coefficient space), which dedupe to higher agreement, and rank-drop slopes
(kernel dim `>= 2`, `all M_i(Z) = 0`), whose valid split locators also dedupe by
the Vandermonde-rank argument above.  The top chart is the only remaining place
where new exact-`A` mass can enter this deficiency-one atlas.

**Degree budget (structural, now checked):** `deg_Z c_i <= t = 128`.
Pseudo-division of `X^n-1` by a degree-`j` Cramer locator takes
`n-j+1` reduction steps.  Each step multiplies by the top coefficient and
increases the slope-degree bound by at most `t`, so every top-chart
pseudo-remainder coefficient has slope-degree at most

```text
(n-j+1)t = (512-128+1)128 = 49280.
```

The rough resultant cap is

```text
nt = 512*128 = 65536,
```

so the pseudo-remainder chart improves the crude eliminant budget by `16256`
degree units.  This degree is still far above `B_Q = 6` before paid-ledger
subtraction; its value is structural: the first filter with actual content
below `A=385`, in packet form, with the paid-ledger subtraction deferred to M4
where it belongs.

### Verified so far (turn 8)

- **Bucket identification.** `regular(A) <=> 2A >= 769` swept over all
  `A in [257, 512]`; `A=385` regular (`t=129 >= j+1=128`), `A=384` not
  (`t=128 < j+1=129`); `A=384` is the maximal underdetermined agreement, with
  deficiency exactly 1; syndrome window `t+j = n-k = 256` is `A`-independent.
- **Toy dichotomy (exhaustive, `F_13`).** One fixed length-8 window `u + z*v`,
  all 13 slopes: the underdetermined `4x5` split (the `A=384` shape) has an
  explicitly constructed and verified kernel vector at **all** 13 slopes
  (certificate vacuous), generically of kernel dimension exactly 1; the regular
  `5x4` split of the *same* windows has full column rank except at few slopes —
  the content of the regular certificate, and its structural loss at `A=384`,
  demonstrated side by side.
- **Cramer kernel vector (exact, `F_13`).** For every toy slope, the signed
  maximal-minor vector `c_i=(-1)^i det(M_i)` satisfies `M c=0`; it is nonzero
  exactly on the full-row-rank chart and vanishes exactly on the rank-drop
  chart.
- **Rank-drop split-locator deduplication (toy subgroup, `F_97`).** For the
  order-8 subgroup and deficiency-one shape `j=t=4`, all degree-4 split
  locators and active-support subpatterns were checked.  Whenever the moment
  Hankel rank drops below `j`, the active-root locator has degree `<j` and
  annihilates the same window, verifying the higher-agreement charge mechanism.
- **Top-chart divisibility filter (`F_13` and `F_97`).** On the `F_13` toy
  pencil, all full-rank top-chart Cramer locators were checked against the
  root test in `H=F_13^*` and against polynomial division by `X^12-1`; the two
  tests agree.  On the order-8 subgroup of `F_97`, all 70 planted split
  degree-4 top-chart cases are Cramer kernels and divide `X^8-1`.  This
  verifies the finite chart criterion used by the pseudo-remainder eliminant.
- **Pseudo-remainder degree budget.** A degree-only pseudo-division recurrence
  verifies that the top-chart pseudo-remainder coefficients have slope-degree
  at most `(n-j+1)t`; at `A=384` this is `49280`, compared with the rough
  resultant cap `65536`.  The script also sweeps small `(n,j,d)` values to
  check the recurrence.
- **Toy pseudo-remainder root table (`F_97`).** For the declared affine family
  interpolating two planted split moment windows, all 97 slopes are classified.
  The root table is exactly `[0, 1]`; there is one rank-drop slope, one
  low-degree slope, and 93 full-rank top-chart nonroots.  The two table roots
  recover the planted root sets, so the pseudo-remainder root-table mechanism
  is replayed end to end on a finite declared family.
- **Root-containment certificate (`F_97[z][X]`).** On the same family, the
  symbolic pseudo-remainder has a nonzero coefficient of degree `19`, below the
  degree cap `20`; its zero set has three slopes and contains the two root-table
  slopes.  This verifies the certificate pattern: one nonzero
  pseudo-remainder coefficient bounds the top-chart root table, while the
  all-zero case must be labelled as a residual/identity chart.
- **Saturated-gcd certificate (`F_97[z][X]`).** Taking the gcd of all four
  nonzero pseudo-remainder coefficients gives a raw degree-7 eliminant with
  roots `[0,1,75]`.  The extra root is on the low-degree boundary:
  `c_j(75)=0`.  Saturating the gcd away from the top Cramer coefficient
  leaves a degree-2 eliminant whose roots are exactly `[0,1]`, matching the
  enumerated full-rank top-chart root table.
- **A=384 residual packet (`F_17^32`).** The verifier emits
  `experimental/data/certificates/hankel-f17-32-m5-underdetermined-a384/f17_32_n512_k256_a384_m5_residual_packet.json`
  and replays it through `scripts/check_aperiodic_eliminant_packet.py`.  The
  packet records a `pivot_atlas` at `A=384`, `j=t=128`: the rank-drop branch and
  low-degree branch are marked `empty` as new exact-`A` mass after
  higher-agreement deduplication, while the full-rank top Cramer locator chart
  is explicitly labelled `residual_obstruction / unknown`.  Its named finite
  object is the top-coefficient-saturated gcd of the pseudo-remainder
  coefficients, with degree cap `49280`.  This is a certified residual packet,
  not a root table.

## Honest scope

- Worst-case bad-slope bounds at `A=384` quantify over all received pairs
  `(u,v)`; like the M3 packets, any eliminant produced here is for **declared
  input families** (printed in the packet) unless and until a lemma removes the
  dependence. No worst-case row claim.
- An eliminant degree `~2^16` does not approach the budget `B_Q = 6`; this
  packet's purpose is the first structural filter and format compliance below
  the regular window, not a threshold claim.
- The top pseudo-remainder root table is not closed here.  It is now explicitly
  represented as a labelled `residual_obstruction / unknown` per §4.6 of the
  roadmap — named, not hidden.
- The committed root table is a toy declared-family table.  It is evidence
  that the mechanism is correctly wired, not evidence that the F17 A384
  root table is small.
