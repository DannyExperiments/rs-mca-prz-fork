## BANKABLE_LEMMA — quotient-action rank

Let
[
A_E=F[X]/(E),\qquad \xi=[X]\in A_E.
]
Define
[
d_M(E):=\deg\minpoly_F(\xi^M).
]

Then
[
\boxed{
d_M(E)
======

# \dim_F F[\xi^M]

\min{\deg m:\ 0\ne m\in F[Y],\ E(X)\mid m(X^M)}.
}
]

### PROOF

A polynomial (m(Y)) annihilates (\xi^M) precisely when
[
m(\xi^M)=0\quad\text{in }A_E,
]
which is equivalent to
[
E(X)\mid m(X^M).
]
The annihilator ideal in (F[Y]) is generated by the minimal polynomial of (\xi^M), and
[
F[\xi^M]\simeq F[Y]/(\minpoly_F(\xi^M)).
]

If (E) is squarefree and split, with root set (R_E\subset F), then
[
\boxed{d_M(E)=|{r^M:r\in R_E}|.}
]
Indeed, (m(\xi^M)=0) iff (m(r^M)=0) for every (r\in R_E).

Thus low (d_M(E)) means that (E) is a **component of a quotient pullback**, not necessarily the whole pullback. Literal pullbacks are only a special case.
