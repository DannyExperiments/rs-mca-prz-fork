## PROOF — exact syndrome transverse-secant correspondence

Let

[

and let

[
\operatorname{syn}:F^L\longrightarrow F^r

[

W_T=\operatorname{span}{h_x:x\in T}.

Because (C) is MDS, every set of at most (r) columns (h_x) is linearly independent.

Fix

[
a=k+\sigma,\qquad j=n-a=r-\sigma.
]

For (f,g\in F^L), write

[
u=\operatorname{syn}(f),\qquad v=\operatorname{syn}(g).
]

For a fixed (j)-set (T\subseteq L), with (S=L\setminus T), the slope (z) is support-wise MCA-bad on (S) if and only if

[
u+zv\in W_T,\qquad v\notin W_T.
\tag{1}
]

### Proof

The word (f+zg) agrees with a codeword on (S) exactly when there is an error vector (e), supported on (T), satisfying

[
\operatorname{syn}(e)=u+zv.
]

This is exactly (u+zv\in W_T).

Likewise, (g) agrees with a codeword on (S) exactly when (v\in W_T). Under (u+zv\in W_T), if (v\in W_T), then

[
u=(u+zv)-zv\in W_T,
]

so both (f) and (g) have degree-(<k) explanations on (S). Conversely, (v\notin W_T) rules out any simultaneous pair. This proves (1).

The same statement remains exact when the original agreement set has size larger than (a). Indeed, suppose (T_0) has size (<j), (u+zv\in W_{T_0}), and (v\notin W_{T_0}). One can extend (T_0) one element at a time to a (j)-set while retaining (v\notin W_T). If every one-element extension contained (v), two such extensions would have intersection (W_{T_0}), by full spark, forcing (v\in W_{T_0}), a contradiction.

Therefore

[
\Gamma(u,v):=
\left{
z\in F:
\exists T\in\binom Lj,\
u+zv\in W_T,\
v\notin W_T
\right}
\tag{2}

is exactly the support-wise MCA-bad slope set at radius (j/n).

**Tangent and common core.** Close distinct-slope pairs force a proper envelope by (14). A fixed minimal envelope contributes at most linearly many internal or independent-union slopes by (18). The external packet (19) is not classified.

**Same witness or support multiplicity.** It cannot create many distinct slopes. The affine syndrome line is injectively parametrized by (z) when (v\ne0), so one syndrome point is one slope. This is why the incidence counterpacket does not refute the corrected theorem.

**Quotient pullback.** The quotient-profile lower contribution is theorem-level. What remains absent is a converse saying that every large split-locator ratio image is quotient-periodic. In particular, quotient removal must be formulated at the witnessed-packet level; a single line may carry both quotient and nonquotient witnesses.

This reduction is proved, but its use requires a repaired arbitrary-word list theorem.

# FULL_SOLVE_ASSESSMENT

The syndrome formulation is exact and is the right global object. It successfully removes the artificial division into (t<\sigma), (t=\sigma), and (t>\sigma).

It does not by itself prove the upper theorem. The unresolved step is not a routine incidence estimate:

* support-incidence counting is false;
* pairwise close-collision forcing is false;
* continuous secant geometry is ambient and therefore vacuous;
* pairwise residue maps are completely surjective on (\sigma)-separated supports;
* ordinary list decoding controls a stronger but still unproved one-dimensional-supercode object;
* extension fields require an additional multiplication-slice theorem.

A route to a full solve exists only through a genuinely higher-order fixed-prime inverse theorem for the complete minor system (28), together with a packet-level quotient classification and control of the external common-envelope branch (19).
