## INTERLEAVED_CONSEQUENCE

### PROOF — linear projection collapses interleaving

Let (K=\mathbb F_q), let (C\subseteq K^n) be any linear code, and let

[

Let (\widehat L) be any scalar upper bound satisfying

[

[
\boxed{
\binom{\widehat L+1}{2}<q.
}
\tag{15}

Then, for every (m\ge1),

[
\boxed{
L_m(\delta)\le\widehat L.
}
\tag{16}
]

To prove this, suppose an interleaved list contained (\widehat L+1) tuples

[
c^{(1)},\ldots,c^{(\widehat L+1)}\in C^m.
]

For (\lambda=(\lambda_1,\ldots,\lambda_m)\in K^m), set

[
\pi_\lambda(c)=\sum_{u=1}^m\lambda_uc_u\in C.

For each pair (c^{(i)}\ne c^{(j)}), the bad projections satisfying

[
\pi_\lambda(c^{(i)})=\pi_\lambda(c^{(j)})

The union of all bad kernels has size at most

[
\binom{\widehat L+1}{2}q^{m-1}<q^m.
]

Choose (\lambda) outside this union. Then all projected codewords are distinct. Every original common agreement position remains an agreement position after projection, so the projected words give (\widehat L+1) distinct scalar codewords in one scalar list, contradicting the scalar bound.

The reverse inequality

[

[
\binom{L_1(\delta)+1}{2}<q,

one has the exact equality

[

\tag{17}

for every arity (m), not merely constant (m).
