
## If PRZ Replies "No"

Ask:

```text
What is the first rejected clause?
```

Then branch:

```text
Extension field rejected:
  Activate Goldilocks prime-field fallback.

Predicate differs:
  Build a predicate adapter or prove the mismatch is fatal.

Sampler differs:
  Need balanced projection or exact pushforward retained mass.

Event filter exists:
  Need exact retained numerator T and denominator M, then check 2^128*T > M.

Small n/domain rejected:
  Need accepted-domain compiler.
```

## If PRZ Is Unsure

Send only the cold packet and ask for a line-by-line admissibility audit:

```text
Please mark each gate ACCEPT / REJECT / UNKNOWN:
row, domain, predicate, sampler, event-retention.
```
