# M1 SPI Dimension-One Eliminant Bound

## Status

PROVED-SYMBOLIC.

This note packages the roadmap node `spi_dim1`: the first underdetermined
dimension-one divisor-pencil chart has a finite eliminant unless it is an
identically valid residual.  For the pinned `F_17^32`, `n=512`, `k=256`,
`A=384` row, the resulting global one-parameter degree cap is `49408`.
The local U1-U5 chart identities used below are isolated in
`experimental/notes/m1/m1_deficiency_one_u1_u5_chain.md`.

## Setup

Let `F` be a field and let `H` be the root set of `X^n-1` in `F`, with
`char(F)` not dividing `n`.  Fix an exact-agreement bucket

```text
t = A-k,        j = n-A,        t = j.
```

Let

```text
M(Z) = M_0 + Z M_1
```

be a deficiency-one `t x (j+1)` split Hankel pencil.  For `0 <= i <= j`,
write `M_i(Z)` for the maximal minor obtained by deleting column `i`, and set

```text
c_i(Z) = (-1)^i M_i(Z),
L_Z(X) = sum_{i=0}^j c_i(Z) X^i.
```

Assume the pencil is not identically rank deficient, so at least one
`c_i(Z)` is nonzero.

## Theorem

Every finite slope lies in one of the following classes.

1. **Rank drop.** All maximal minors vanish.  Since every entry of `M(Z)` is
   affine-linear in `Z`, every `c_i(Z)` has degree at most `t`; hence the
   rank-drop slopes are contained in the roots of any chosen nonzero maximal
   minor, a set of size at most `t`.

2. **Full-rank low-degree chart.** The matrix has rank `t` and `c_j(z)=0`.
   The Cramer kernel locator has degree `< j`, so it is charged to a
   higher-agreement contained branch rather than to a new exact-`A` degree-`j`
   locator.

3. **Full-rank top chart.** The matrix has rank `t` and `c_j(z) != 0`.
   The kernel is generated by the degree-`j` locator `L_z`.  Since
   `X^n-1` is squarefree with root set `H`, this locator is valid exactly when

   ```text
   L_z(X) | X^n - 1.
   ```

   Pseudo-divide

   ```text
   c_j(Z)^delta (X^n - 1) = Q(Z,X)L_Z(X) + R(Z,X),
   delta = n-j+1,
   R(Z,X) = sum_{m=0}^{j-1} rho_m(Z) X^m.
   ```

   The pseudo-division recurrence uses `delta` multiplication/subtraction
   steps, and each coefficient `c_i(Z)` has degree at most `t`.  Therefore

   ```text
   deg_Z rho_m <= delta t = (n-j+1)t.
   ```

Thus exactly one of the following holds.

```text
(E) Some rho_m is nonzero.
    Then all new finite exact-A degree-j split-locator slopes are contained in
    the zero set of a nonzero eliminant of degree at most

        t + (n-j+1)t.

(V) All rho_m vanish identically.
    Then the top chart is an identically valid divisor-pencil residual.  This
    is not hidden inside the degree bound; it is a named residual branch that
    must be paid, refuted, or structurally classified separately.
```

For `n=512`, `k=256`, `A=384`, we have `t=j=128` and `delta=385`, so

```text
top pseudo-remainder coefficient degree <= 385*128 = 49280,
global dimension-one eliminant cap        <= 128 + 49280 = 49408.
```

## Proof

The signed maximal-minor vector is the standard Cramer kernel vector for a
full-rank `t x (t+1)` matrix.  If all maximal minors vanish, the matrix has
rank `< t`, giving the rank-drop chart.  If not, the kernel is one-dimensional.
When `c_j=0`, its locator has degree `< j`, so it is not a new exact
degree-`j` contribution.

On the top chart, `c_j != 0`, so the kernel locator has degree `j`.  Because
`X^n-1` is squarefree under `char(F) not | n`, a degree-`j` locator splits on
`H` exactly when it divides `X^n-1`.  Pseudo-division clears the top
coefficient and gives an equivalent divisibility test on the top chart.  The
degree bound follows by induction through the pseudo-division recurrence: each
step multiplies existing coefficients by the leading coefficient and subtracts
a multiple of `L_Z`; both operations increase `Z`-degree by at most `t`.

If at least one pseudo-remainder coefficient is nonzero, the valid top-chart
slopes are contained in its zero set, which has size at most `(n-j+1)t`.  The
rank-drop chart contributes at most `t` further finite slopes.  Adding these
two one-dimensional algebraic caps gives `t + (n-j+1)t`.  If every
coefficient vanishes identically, no finite eliminant exists on the top chart;
that is precisely the identically valid residual case.

## Scope

This note proves a dimension-one eliminant-or-residual reduction.  It does not
compute the `F_17^32` root table, does not close SPI in higher dimension, and
does not classify identically valid residual pencils.  Its role is to remove
the first rung of the SPI/deficiency ladder from the fuzzy target list: the
dimension-one case is now a concrete bounded eliminant problem unless a named
residual appears.
