## What This Proves

The round supports the following source-level adapter direction:

1. Every official bad slope can first be routed into an endpoint or interior
   source object.
2. Endpoint contributes at most one slope.
3. Interior objects must be stratified by intrinsic denominator degree into
   low, balanced, or high branches.
4. High branches have a transverse affine-plane inverse/normal form.
5. Balanced branches have a faithful complement-locator or syndrome-pencil
   formulation.
6. A q-ledger proof is possible only after all branch caps and same-field slope
   injections are explicit.

This is official-route relevant because it works with official source objects
and actual line slopes, not only finite Cycle84 model products.

## What It Does Not Prove

It does not prove:

- the official residual numerator is polynomially bounded;
- `N_off / q_line <= 2^-128`;
- source `AP_corr` descends to the Gate B rank-escape/separator hypothesis;
- all low, balanced, and high branches have polynomial chart covers;
- all endpoint, quotient, periodic, contained, tangent, field, affine-color,
  same-slope, and internal-value charges are capped without double counting;
- the protocol challenge field `q_chal` can replace the line field `q_line`;
- a source-valid counterpacket family exists.

## Field Ledger Discipline

The Pro returns converged on the same field discipline:

- `q_gen` belongs to generated-field entropy and corrected-reserve hypotheses.
- `q_code` is the code alphabet or scalar-extension source field.
- `q_line` is the field from which MCA slopes are sampled; it is the denominator
  of the official bad-slope probability.
- `q_chal` is irrelevant unless a separate protocol-transfer theorem identifies
  the challenge experiment with the line-slope experiment.
- The final security inequality is `N_off / q_line <= 2^-128`, equivalently
  `N_off <= floor(q_line / 2^128)`.

No return legitimately pays the final denominator with `q_gen`, `q_code`, or
`q_chal`.

## First Failure Line

The older reduction fails at:

`official bad slope -> tau(g) = sigma -> one complement-line U`

Primitive directions with intrinsic denominator degree `sigma + 1` cut that
line.

After the repair by intrinsic denominator degree, the first open line is:

`LOW / BALANCED / HIGH interior source object -> polynomial charged or tagged Gate B cover`

Equivalently, prove a theorem that every uncharged interior object either:

- is paid by an explicit official charge; or

  the required AP/rank-escape/separator condition.
