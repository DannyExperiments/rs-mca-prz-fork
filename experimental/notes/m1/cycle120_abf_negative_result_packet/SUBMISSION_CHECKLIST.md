# Submission Checklist

This checklist is based on the Proximity Prize website as checked on
2026-06-24.

## Required Main PDF

The main PDF must:

```text
1. State the claimed result.
2. Explain how it relates to the prize challenge.
3. Situate the result with respect to prior work.
4. Clearly label supplementary material.
```

For this packet, the main PDF should state:

```text
Assuming the Cycle84/Cycle116 proof chain, the ABF grand MCA definition
admits a smooth RS row C=RS[F_17^32,H,256] with epsilon_mca(C,125/256)>2^-128.
```

It should not state:

```text
exact delta*_C solved;
ordinary list decoding solved;
protocol soundness broken;
peer-reviewed prize accepted;
prize awarded.
```

## Public Timestamp

The website says authors must make the submission public on an open academic
repository such as:

```text
IACR ePrint
arXiv
```

The first public version is the formal timestamp. Therefore:

```text
Do not publish a messy raw packet as the first public version.
Prepare a clean human-edited PDF first.
```

## Peer Review

The website says submissions are considered only after scientific peer review
in a reputable field-appropriate conference or journal.

Immediate implication:

```text
The next public artifact can be a timestamped preprint, but final prize
consideration still requires peer-reviewed acceptance unless the judges
exercise an exception.
```

## AI Policy

The website permits AI-aided submissions, but expects:

```text
human verification,
human editing,
standard field language and notation,
human authors solely responsible for correctness.
```

Immediate implication:

```text
The PDF must be edited by a human author before submission.
Model outputs and checker logs should remain supplementary, not the main voice.
```

## Partial Progress And Splitting

The website says:

```text
partial progress is encouraged;
awards may be split among multiple submissions;
any award is shared equally among named authors unless otherwise specified.
```

Immediate implication:

```text
If the theorem survives review but does not determine exact delta*_C,
it can still be framed as prize-facing partial progress.
Authorship and contribution statements should be decided before public upload.
```
