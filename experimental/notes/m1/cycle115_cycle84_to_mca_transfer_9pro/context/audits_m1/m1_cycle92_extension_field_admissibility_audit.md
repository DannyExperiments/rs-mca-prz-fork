```text
L-CYCLE92-EXTENSION-FIELD-POWER-BASIS-COVERAGE
```

`H <= F_q^*` be the order-`N` subgroup generated by a primitive `N`th root
`omega`. Then `F_p(omega)=F_q`.

For `k=rho N`, `ell=k+1`, and the monomial line

```text
f(x)=x^{k+1},   g(x)=x^k,
```

the noncontained bad slopes include the restricted sumset:

```text
ell^wedge H.
```

The power-basis sign-digit slice gives:

```text
|ell^wedge H|
  >= sum_{w <= min(ell,d), w == ell mod 2} binom(d,w) 2^w.
```

```text
d = ord_256(5) = 64

```

```text

  = 180,796,807,614,761,533,440

```

