# BANKABLE_LEMMA

The finite target can be made exact, but no rate-only value of (\delta_C^*(2^{-128})) follows from the current banked results.

The main correction to the proposed ledger is:

[
\boxed{
U_{\mathrm{MCA}}
================

\max!\left{
U_{\mathrm{res}},
U_{\mathrm{high}},
N_{\mathrm{occ}}+U_{\mathrm{tan}}+U_{\mathrm{quot}}+U_{\mathrm{ap}}
\right}.
}
]

Residual, balanced, and high-denominator estimates are alternative line branches, not additive phenomena on every line. The requested sum

[
N_{\mathrm{total}}
==================

N_{\mathrm{occ}}
+U_{\mathrm{tan}}
+U_{\mathrm{quot}}
+U_{\mathrm{res}}
+U_{\mathrm{high}}
+U_{\mathrm{ap}}
]

is nevertheless a valid, deliberately conservative certificate because
(U_{\mathrm{MCA}}\le N_{\mathrm{total}}).

## 1. Exact finite certification theorem

Let

[
B\subseteq F,\qquad |B|=q_{\mathrm{gen}},\qquad |F|=Q,
]

where (B) is the field generated by the smooth evaluation domain
(L\subseteq B), and let

[
C=\operatorname{RS}[F,L,k],\qquad n=|L|,\qquad r=n-k.
]

Fix an integer agreement threshold

[
a=k+\sigma,\qquad j=n-a=r-\sigma,
]

and define the exact target numerator

[
\boxed{T_Q:=\left\lfloor \frac{Q}{2^{128}}\right\rfloor.}
]

Assume initially that (\sigma\ge1). Let (\mathsf H) be an (r\times n) parity-check matrix for (C), with column (h_x) at (x\in L), and put

[
V_T:=\operatorname{span}_F{h_x:x\in T}.
]

### Exact MCA numerator

For an affine syndrome line

[
\ell(z)=u+zv,\qquad u,v\in F^r,
]

define

[
\operatorname{Bad}_j(\ell)
==========================

\left{
z\in F:
\exists T\in\binom Lj,\
\ell(z)\in V_T,\
v\notin V_T
\right}.
]

Then the exact scalar support-wise MCA numerator is

[
\boxed{
M_C(a)=
\max_{u,v\in F^r}
|\operatorname{Bad}_j(u+Fv)|
}
]

and

[
\epsilon_{\mathrm{mca}}!\left(C,1-\frac an\right)
=================================================

\frac{M_C(a)}{Q}.
]

Consequently,

[
\boxed{
M_C(a)\le T_Q
\quad\Longrightarrow\quad
\epsilon_{\mathrm{mca}}!\left(C,1-\frac an\right)
\le 2^{-128}.
}
]

### Exact interleaved-list numerator

For (m\ge1), let (\mathcal E_{m,j}) be the set of (m\times n) error matrices with at most (j) nonzero columns. For rows (e_1,\ldots,e_m), define

[
\Psi_m(E)=
\bigl(\mathsf He_1^{\mathsf T},\ldots,
\mathsf He_m^{\mathsf T}\bigr)\in(F^r)^m.
]

Then

[
\boxed{
L_{C,m}(a)
==========

\max_{s\in(F^r)^m}
\left|
\Psi_m^{-1}(s)\cap\mathcal E_{m,j}
\right|
}
]

is exactly the worst (m)-interleaved list size at agreement at least (a). Hence

[
\boxed{
L_{C,m}(a)\le T_Q
\quad\Longrightarrow\quad
|\Lambda(C^{=m},1-a/n)|
\le 2^{-128}Q.
}
]

These two boxed formulas are exhaustive finite checkers. They use no denominator decomposition and automatically include (t<\sigma), (t=\sigma), (t>\sigma), and every overagreement layer.

## 2. The exact six-term MCA certificate

A modular proof may classify a line by its intrinsic minimal residue-denominator degree (\tau(v)). This must be the **minimal** degree: raw denominator degree cannot be used because denominators can be padded upward.

For a smooth multiplicative domain, define (\tau(v)) as the least (t) for which a representative direction word (g) admits

[
g(x)=R(x)-\frac{B_{\mathrm{num}}(x)}{E(x)}
\quad(x\in L),
]

with

[
\deg R<k,\qquad
\deg E=t,\qquad
\deg B_{\mathrm{num}}<t,\qquad
E|_L\ne0.
]

The degree is invariant under adding codewords to (g).

### 2.1 Occupancy/main term

At the fixed target (a=k+\sigma), write

[
N_\sigma=\binom nj=\binom n{k+\sigma}.
]

The denominator-free line-occupancy baseline is

[
\boxed{
N_{\mathrm{occ}}(\sigma)
========================

\left\lceil
\frac{N_\sigma}{Q^{\sigma-1}}
\right\rceil.
}
]

This is the correct fixed-radius term. The expression

[
\binom n{k+t}/Q^{t-1}
]

is the same formula when the radius itself is balanced at (a=k+t); it should not be summed over every denominator degree (t) at one fixed reserve (\sigma).

In the standard dual-GRS/Hankel coordinates, the transverse landing count has the exact zero-mode decomposition

[
R(\ell)
= ======

## \frac{N_\sigma}{Q^{\sigma-1}}

QK_0(\ell)
+
\operatorname{Err}(\ell),
]

where (K_0(\ell)) counts locators lying in the common kernel of the pencil. Since the common-kernel term is nonpositive,

[
R(\ell)
\le
N_{\mathrm{occ}}(\sigma)
+
\operatorname{Err}(\ell)_+.
]

Thus (N_{\mathrm{occ}}) is an exact Fourier/character zero-mode baseline. What remains unproved is the required uniform one-sided control of the nonzero modes.

### 2.2 Tangent/common-envelope term

A finite certificate must give a machine-checkable predicate identifying tangent or common-envelope witnesses and an integer bound

[
|\operatorname{Bad}*{\mathrm{tan}}(\ell)|
\le U*{\mathrm{tan}}.
]

One banked partial estimate is the following. If

[
\operatorname{span}(u,v)\subseteq V_R,
\qquad |R|=j+d<r,
]

with (R) minimal, then witnesses satisfying (|R\cup T|\le r) contribute at most

[
\left\lfloor\frac{j+d}{d}\right\rfloor
\le j+1
]

distinct slopes. The external-envelope witnesses with

[
|R\cup T|>r
]

are not bounded by this argument and must appear explicitly inside (U_{\mathrm{tan}}).

A vague “common-envelope exception” is not a certificate. PRZ needs a membership predicate, an overlap convention, and a distinct-slope bound.

### 2.3 Quotient/action-rank term

For a balanced residue chart (E), the relevant invariant is

[
d_M(E)
======

\deg\minpoly_F([X^M]\bmod E).
]

If (E) is squarefree and split, this is the number of distinct values (r^M) among roots (r) of (E).

A finite quotient certificate consists of a registry indexed by quotient orders and defects,

[
(M,d,\text{template type})\longmapsto U_{M,d,\mathrm{type}},
]

together with a proof that every quotient-attributed witness belongs to one registry entry. Then

[
U_{\mathrm{quot}}
=================

\sum_{M,d,\mathrm{type}}U_{M,d,\mathrm{type}}
]

is a valid union bound.

Two restrictions are essential:

1. The syntactic exclusion (E\ne E_0(X^M)) is inadequate.
2. The smaller canonical charge
   [
   2^{(\beta(\rho)/H_2(\rho))\mathcal Q_H}
   ]
   is false for arbitrary-anchor all-line MCA.

The punctured-fiber construction forces the quotient budget to allow at least the full profile scale

[
2^{\mathcal Q_H},
]

but even this is currently only a necessary scale, not a proved universal upper bound.

### 2.4 Residual actual-list term

For (t<\sigma), every bad slope injects into an actual list for the shifted code

[
\operatorname{RS}[F,L,k+t]
]

at the same agreement (a=k+\sigma). Define

[
\mathsf L_F(k+t,a)
==================

\max_w
\left|
\left{
Q\in F[X]_{<k+t}:
|{x:Q(x)=w(x)}|\ge a
\right}
\right|.
]

Then

[
|\operatorname{Bad}_j(\ell)|
\le \mathsf L_F(k+t,a)
\qquad\text{when }\tau(v)=t<\sigma.
]

Therefore the exact residual input is

[
\boxed{
U_{\mathrm{res}}
================

\max_{1\le t<\sigma}U_{\mathrm{list}}(k+t,a),
}
]

where (U_{\mathrm{list}}(k+t,a)) is a proved upper bound on the **actual** list.

The residual entropy surplus is

[
(\sigma-t)\log_2q_{\mathrm{gen}}
--------------------------------

\log_2\binom n{k+\sigma},
]

not the original (\sigma)-surplus. Thus a theorem at reserve (\sigma) does not automatically pay the residual branch near (t=\sigma).

Raw feasible support fibers cannot be substituted here.

### 2.5 High-denominator affine-plane term

For (t=\sigma+h>\sigma), a support (S), (|S|=a), gives the exact residue-plane incidence

[
z[B_{\mathrm{num}}]_E
\in
[I_S(w)]*E
+
[L_SF[X]*{<h}]_E,
]

with noncontainment equivalent to

[
[B_{\mathrm{num}}]*E
\notin
[L_SF[X]*{<h}]_E.
]

The affine plane has dimension (h) inside the (t)-dimensional algebra, hence codimension (\sigma).

The finite input is an integer (U_{\mathrm{high}}) satisfying

[
|\operatorname{Bad}*j(\ell)|
\le U*{\mathrm{high}}
\qquad
\text{for every line with }\tau(v)>\sigma.
]

This term must be restricted to **minimal or primitive** high-denominator lines. Without minimality, the (t=r) residue chart represents every syndrome line, making the term equal to the entire unsolved MCA numerator.

### 2.6 Aperiodic transverse-secant term

After tangent and quotient witnesses are removed by explicit predicates, let

[
R_{\mathrm{ap}}(\ell)
=====================

#\left{
(z,T):
T\text{ is untagged},
\ell(z)\in V_T,
v\notin V_T
\right}.
]

A sufficient finite hypothesis is

[
\boxed{
R_{\mathrm{ap}}(\ell)
\le
N_{\mathrm{occ}}(\sigma)+U_{\mathrm{ap}}
\quad\text{for every relevant line }\ell.
}
]

This is stronger than a distinct-slope theorem, because it bounds landings rather than their value set, but it is exact and sufficient.

Alternatively PRZ may consume a direct distinct-slope statement

[
|\operatorname{Bad}*{\mathrm{ap}}(\ell)|
\le N*{\mathrm{occ}}+U_{\mathrm{ap}}.
]

The certificate must say which version is proved.

### Finite MCA conclusion

Under the preceding branch and coverage hypotheses,

[
\boxed{
M_C(a)
\le
U_{\mathrm{MCA}}(\sigma)
:=
\max\left{
U_{\mathrm{res}},
U_{\mathrm{high}},
N_{\mathrm{occ}}
+U_{\mathrm{tan}}
+U_{\mathrm{quot}}
+U_{\mathrm{ap}}
\right}.
}
]

Consequently,

[
U_{\mathrm{MCA}}(\sigma)\le T_Q
]

certifies MCA safety.

The user-requested additive ledger

[
\boxed{
N_{\mathrm{total}}
==================

N_{\mathrm{occ}}
+U_{\mathrm{tan}}
+U_{\mathrm{quot}}
+U_{\mathrm{res}}
+U_{\mathrm{high}}
+U_{\mathrm{ap}}
}
]

is a simpler sufficient test:

[
\boxed{
N_{\mathrm{total}}\le T_Q
\quad\Longrightarrow\quad
\epsilon_{\mathrm{mca}}(C,1-a/n)\le2^{-128}.
}
]

The nested maximum is the stronger correct finite alternative.

## 3. Exact list and interleaving certificate

The six MCA terms are not themselves a list-size decomposition. The target list problem has its own exact sparse-syndrome fiber.

Let

[
C_B=\operatorname{RS}[B,L,k],
\qquad d=[F:B].
]

Coordinate expansion in a (B)-basis gives exact column-agreement-preserving identifications

[
C\cong C_B^{=d},
\qquad
C^{=m}\cong C_B^{=dm}.
]

Let (L_B(a)) be the worst scalar list size for (C_B) at agreement (a).

### Finite projection theorem

If an integer (U_B) satisfies

[
L_B(a)\le U_B
]

and

[
\boxed{
\binom{U_B+1}{2}<q_{\mathrm{gen}},
}
]

then for every extension degree and every interleaving arity,

[
\boxed{
L_{C,m}(a)\le U_B.
}
]

Therefore the scalar certificate

[
U_B\le T_Q,
\qquad
\binom{U_B+1}{2}<q_{\mathrm{gen}}
]

certifies

[
|\Lambda(C^{=m},1-a/n)|\le2^{-128}Q.
]

If the projection inequality fails, a scalar theorem is insufficient. PRZ then needs a direct bound on

[
\max_s
|\Psi_m^{-1}(s)\cap\mathcal E_{m,j}|.
]

In the same-field case (q_{\mathrm{gen}}=Q), and under the official cap (Q\le2^{256}), the target-level condition is automatic:

[
\binom{T_Q+1}{2}<Q.
]

### Correct scalar list object

For a scalar syndrome (s), define

[
\nu_e^\circ(s)
==============

#\left{
E\subseteq L:
|E|=e,
s=\sum_{x\in E}c_xh_x
\text{ with every }c_x\ne0
\right}.
]

Then

[
\boxed{
L_B(a)
======

\max_s\sum_{e=0}^{j}\nu_e^\circ(s).
}
]

Thus the finite list theorem must:

* count full error supports with nonzero coefficients;
* sum every error weight (e\le j);
* include quotient overagreement layers;
* include common-core/tangent lists;
* bound primitive block-secant kernels.

A raw count of feasible (a)-subsets is invalid.

## 4. Proof details

### Syndrome transverse-secant equivalence

A word (f+zg) agrees with a codeword outside (T) exactly when its syndrome is the syndrome of an error supported on (T):

[
f+zg-c=e,\quad\operatorname{supp}(e)\subseteq T
\quad\Longleftrightarrow\quad
u+zv\in V_T.
]

Under this incidence, (v\in V_T) implies (u\in V_T), so both anchor and direction have local codeword explanations. Hence noncontainment is exactly (v\notin V_T).

If (|T|<j), the MDS/full-spark property permits extending (T) to a (j)-set while preserving (v\notin V_T). Therefore exact (j)-sets suffice when (\sigma\ge1).

For a fixed (T) with (v\notin V_T), the affine line intersects (V_T) in at most one point. Hence, unconditionally,

[
\boxed{
M_C(a)\le\min\left{Q,\binom nj\right}.
}
]

This improves the cruder tail-of-binomial support bound for MCA.

### Sparse-syndrome list equivalence

For a received (m)-row word (Y) and codeword tuple (P), the error (E=Y-P) has at most (j) nonzero columns precisely when (P) agrees with (Y) on at least (a) columns. Its row syndromes equal the fixed syndrome tuple of (Y).

Conversely every sparse error with that syndrome gives exactly one codeword tuple. This proves the sparse-fiber formula.

Because every set of at most (r) parity-check columns is independent, for any fixed column support (E), every row’s error values are uniquely determined. Consequently,

[
\boxed{
L_{C,m}(a)
\le
\sum_{e=0}^{j}\binom ne
}
]

for every (m), independently of interleaving arity.

This gives a completely unconditional finite safe-side certificate when the right-hand side is at most (T_Q).

### Projection proof

Suppose (U_B+1) interleaved codeword tuples lie in one list. For each pair of distinct tuples, the coefficient vectors
(\lambda\in B^{dm}) that make their scalar projections equal form a proper linear subspace of size at most (q_{\mathrm{gen}}^{dm-1}).

The union of all pair-collision subspaces has size at most

[
\binom{U_B+1}{2}q_{\mathrm{gen}}^{dm-1}
<
q_{\mathrm{gen}}^{dm}.
]

Choose a projection outside this union. All projected codewords are distinct, while every common agreement column remains an agreement after projection. This would produce (U_B+1) scalar listed codewords, contradicting the scalar bound.

### Threshold rounding

For either numerator (N_X(a)), define

[
a_X^*
=====

\min{a:N_X(a)\le T_Q},
\qquad
\sigma_X^*=a_X^*-k.
]

The exact largest safe grid radius is

[
\boxed{
\delta^{\mathrm{grid}}_{X,\max}
===============================

# \frac{n-a_X^*}{n}

1-\rho-\frac{\sigma_X^*}{n}.
}
]

For arbitrary real (\delta), with agreement threshold
(a(\delta)=\lceil(1-\delta)n\rceil), safety holds exactly for

[
\delta<
\boxed{
1-\frac{a_X^*-1}{n}
===================

1-\rho-\frac{\sigma_X^*-1}{n}.
}
]

Thus the frequently displayed (+1/n) is the transition point or supremum of safe radii. The endpoint itself is unsafe when (a_X^*) is minimal. This endpoint convention must be recorded in any claimed (\delta_C^*) formula.

If (Q<2^{128}), then (T_Q=0). Both numerators are at least (1) at radius zero, so no radius is safe.

## 5. Asymptotic entropy-only candidate

This is a corollary only under the additional assumption that all structural terms are negligible relative to (T_Q).

Let

[
b_{\mathrm{line}}=\log_2Q,
\qquad
b_{\mathrm{gen}}=\log_2q_{\mathrm{gen}},
\qquad

The exact finite occupancy-only reserve is

\le
T_QQ^{\sigma-1}
\right}.

Writing

[

[

For fixed rate and subexponential fields,

[
\eta
\sim
\frac{H_2(\rho)}{b_{\mathrm{line}}}.

## 6. Why a rate-only formula is impossible

Even at a fixed official rate, the finite threshold depends on:

[
Q,\ q_{\mathrm{gen}},\ n,\ \gcd(n,k),
]

the full quotient divisor profile, the extension degree, the domain, and the actual line/list concentration constants.

Three direct obstructions suffice:

1. If (Q<2^{128}), no radius is safe; if (Q\ge2^{128}), radius zero is safe.
2. MCA has the necessary tangent condition
   [
   j=n-k-\sigma\le T_Q.
   ]
   Thus (Q/n) affects the threshold independently of entropy.
3. Quotient packets depend on divisibility. For example, under (Q<2^{256}), the list packets
   [
   \binom{255}{128},\quad
   \binom{255}{64},\quad
   \binom{255}{32},\quad
   \binom{511}{32}
   ]
   all exceed (2^{128}), producing reserve floors (n/256) for the first three rates and (n/512) for rate (1/16), but only when the corresponding quotient divides (n).

The strongest correct finite alternative is therefore the code-specific integer reserve

[
\boxed{
\sigma_{X,C}^*
==============

\min{\sigma:N_X(k+\sigma)\le T_Q},
}
]

together with the exact safe-grid radius and transition formula above.

To certify the exact threshold rather than merely one safe radius, PRZ needs:

* an upper certificate at (\sigma^*);
* an explicit failure witness with numerator (>T_Q) at (\sigma^*-1).

Monotonicity then closes the entire threshold.

## 7. Terms and constants still unproved

The following are not presently available with prize-usable finite constants.

1. **Aperiodic discrepancy**
   [
   U_{\mathrm{ap}}(n,k,Q,\sigma).
   ]
   No uniform finite one-sided bound is proved for quotient-free, envelope-free syndrome lines.

2. **External common-envelope contribution.**
   The internal minimal-envelope bound is known; the external branch
   (|R\cup T|>r) remains open.

3. **Universal quotient/action-rank upper classifier.**
   The full quotient profile is a necessary lower scale, not a proved upper classification.

4. **Residual actual-list bounds**
   [
   U_{\mathrm{list}}(k+t,a)
   ]
   for every (t<\sigma), particularly (t=\sigma-1).

5. **Primitive high-denominator affine-plane bound**
   [
   U_{\mathrm{high}}.
   ]

6. **Target scalar full-support list theorem**
   [
   \max_s\sum_{e\le j}\nu_e^\circ(s).
   ]

7. **Extension-valued MCA transfer.**
   Unlike lists, no scalar projection theorem currently reduces arbitrary (F)-syndrome lines to scalar (B)-lines.

8. **Explicit constants replacing (n^{1+o(1)}).**
   An unspecified asymptotic exponent cannot be compared with (T_Q).

## 9. Route-board impact

This banks four points.

* The finite mathematical target is now exact and integer-valued.
* The requested additive six-term ledger is valid, but the sharper structure is a maximum across denominator branches.
* Interleaved lists reduce exactly to scalar lists only after the finite projection inequality is verified.
* A rate-only (\delta_C^*) formula is cut; the strongest valid target is a code-specific reserve certificate.

The global syndrome route remains preferable. A successful denominator-free theorem would absorb both (U_{\mathrm{res}}) and (U_{\mathrm{high}}), leaving only occupancy, explicit template budgets, and aperiodic discrepancy.
