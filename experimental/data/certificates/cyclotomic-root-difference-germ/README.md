# Cyclotomic Root-Difference Germ

This directory contains the replayable arithmetic packet for
`experimental/notes/thresholds/cyclotomic_root_difference_germ.md`.

Replay:

```text
python3 experimental/scripts/verify_cyclotomic_root_difference_germ.py
```

Regenerate:

```text
python3 experimental/scripts/verify_cyclotomic_root_difference_germ.py --emit
```

The packet verifies the closed norm formula against exact cyclotomic
resultants and records the unit/sign template counts for small quotient
orders.
