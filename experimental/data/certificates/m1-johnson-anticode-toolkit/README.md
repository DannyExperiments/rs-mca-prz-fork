# Johnson Anticode Toolkit

This directory contains the replayable certificate for
`experimental/notes/m1/m1_johnson_anticode_toolkit.md`.

Replay:

```text
python3 experimental/scripts/verify_m1_johnson_anticode_toolkit.py
```

Regenerate:

```text
python3 experimental/scripts/verify_m1_johnson_anticode_toolkit.py --emit
```

The packet verifies exact small Johnson-graph packing cases, the binomial
density identity, and the one-step mixing scope table used by the XR notes.
