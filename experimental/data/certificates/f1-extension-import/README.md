# F1 extension-import certificate

This directory contains the deterministic JSON artifact emitted by:

```bash
python3 experimental/scripts/verify_f1_extension_import_lemma.py --emit
```

The packet verifies the `ext_import` arithmetic: base-rational nondegenerate
pencils have only base-field isolated slopes, and the extension-pole numerator
crosses a denominator budget at the exact printed list-size threshold.
