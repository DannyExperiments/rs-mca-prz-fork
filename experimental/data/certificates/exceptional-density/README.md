# Exceptional Density

This directory contains the replayable arithmetic packet for
`experimental/notes/thresholds/exceptional_density.md`.

Replay:

```text
python3 experimental/scripts/verify_exceptional_density.py
```

Regenerate:

```text
python3 experimental/scripts/verify_exceptional_density.py --emit
```

The packet factors small cyclotomic `e_1` difference norms exactly and emits
official-rate dyadic incidence bounds for large-prime collision mass.
