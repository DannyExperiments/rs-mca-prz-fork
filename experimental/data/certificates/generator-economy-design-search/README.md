# Generator-Economy Design Search Certificate

This directory stores the E12/QA.15 toy generator-economy search emitted by:

```bash
python3 experimental/scripts/verify_generator_economy_design_search.py --emit
```

The certificate counts unit/sign difference templates for cyclic-orbit unions
of `ell`-subset centers in `Z/NZ`, exactly at `N=16` and with deterministic
sampled orbit representatives at `N=32`.
